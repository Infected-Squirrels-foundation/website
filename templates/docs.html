WTF is this

Baud●tC●de is a programming language compatible with Baudot Code. It's a fantasy what would Émile Baudot invent would he need an SSH connection to Babbage machine or so..

Syntax

Allowed symbols

Only allowed symbols in Baud●tC●de are 0 and 1.

Program structure

Program structure in Baud●tC●de is freaking simple — it's just a sequence of opertaions delimited by nothing. Each operaion consists of an opcode and operand. Opcodes and operands are coded in 5 bits (Baudot Code compatibility). Example of a single operation 
0000100010this operation adds value from stack with 00010 value
First 5 bits (00001) is addition opcode, last 5 bits if binary representation of digit "2".

Memory

To store data during runtime you have 3 registers and stack. Each register can store one value at a time (5 bits). Stack can store up to 200 values.

Registers

As stated before there are 3 registers avalible to Baud●tC●de program: A, B and C. Note whenever you need to adress a register you have to use valid Baudot code value: 00100 for A, 01001 for B and 01101 for C.
